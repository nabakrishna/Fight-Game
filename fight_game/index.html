Python Fighting Game

2D local multiplayer fighter made with Python + Pygame

How to Play

Player 1

  • Move: A / D
  • Jump: W
  • Crouch: S
  • Dash: Left Shift
  • Block: H
  • Light: F
  • Heavy: G

Player 2

  • Move: Left / Right
  • Jump: Up
  • Crouch: Down
  • Dash: Right Shift
  • Block: ; (Semicolon)
  • Light: K
  • Heavy: L

Goal: Reduce the opponent's health to 0 or have more health when the timer ends. Best of 3 rounds.

Game

If the game is blank, wait a few seconds for WebAssembly to load.

Built with Python, Pygame, and pygbag.
